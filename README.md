# Python Chess ♟️

A Python-based chess game with a graphical user interface built using **Tkinter**.
This project was developed as a personal learning and portfolio project.

## Features
- GUI chessboard (Tkinter)
- Player vs Player and AI-supported modes
- Legal move highlighting
- Move log
- Pawn promotion, castling, en passant
- Undo and restart functionality
- Check and checkmate detection

## How to Run

```bash
python chess.py
